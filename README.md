# PelosiTrader - Nancy Pelosi's Stock Trade Tracker 🛹

Streamlit App zur Echtzeit-Nachverfolgung aller Trades von Nancy Pelosi.

## Setup
```bash
pip install -r requirements.txt
streamlit run main.py
```

## Features
- Live Trade Feed (scraped von QuiverQuant)
- Echtzeit-Kursdaten via Yahoo Finance
- Portfolio Performance Analyse
- Excess Return Tracking
